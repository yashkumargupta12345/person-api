## ðŸ“¦ Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/your-username/person-manager.git
   cd person-manager
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a .env file using the template:
   ```bash
   cp .env.example .env
   ```
4. Add your MongoDB connection string in .env:
   ```bash
   MONGO_URI=your-mongodb-uri
   ```
5. Run the server:
   ```bash
   node app.js
   ```
6. Open browser at:
   ```bash
   http://localhost:3000/
   ```
